## Música

La página ya tiene una música demo creada con JavaScript, así que funciona aunque no agregues ningún MP3.

Si quieres usar un audio real, coloca un archivo llamado exactamente:

```txt
assets/music/clem.mp3
```

El navegador solo reproducirá música después de que la persona presione el botón de play.

## GitHub Pages

Sube el contenido de esta carpeta directamente a tu repositorio. El archivo `index.html` debe quedar en la raíz, no dentro de otra carpeta.
